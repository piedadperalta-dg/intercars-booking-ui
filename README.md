# Intercars – Car Rental Booking UI

## 🇺🇸 Overview
UI/UX redesign for a Miami-based car rental company, focused on improving the digital presence and enabling a complete online booking experience.  
This project was developed as part of my work at Kurve Studio.

## 🇪🇸 Resumen
Rediseño UI/UX para una empresa de renta de vehículos con base en Miami, enfocado en mejorar su presencia digital y permitir un flujo completo de reservas online. 
Este proyecto fue desarrollado como parte de mi trabajo en Kurve Studio.

---

## 🇺🇸 My Role
UI/UX Designer at Kurve Studio  
- Interface redesign  
- User flow optimization  
- Booking experience definition  

## 🇪🇸 Mi rol
Diseñadora UI/UX en Kurve Studio  
- Rediseño de interfaz  
- Optimización de recorridos  
- Definición del flujo de reserva   

---

## 🇺🇸 Problem
The original website presented several limitations:
- No online booking flow  
- Low visual consistency  
- Unclear navigation and content structure  
- Lack of clarity in pricing and reservation steps  

## 🇪🇸 Problema
La web original presentaba varias limitaciones:
- Falta de flujo de reserva online  
- Baja consistencia visual  
- Navegación poco clara  
- Falta de claridad en precios y pasos del proceso  

---

## 🇺🇸 Solution

### Booking Flow
- Step-by-step reservation process  
- Clear progress indication  
- Transparent pricing and included features  
- Reduced cognitive load  

### UI System
- Consistent visual language across all pages  
- Defined hierarchy for readability  
- Reusable components  

### Navigation & Content
- Multiple entry points to booking (CTA-driven)  
- Clear sections: vehicle types, offers, benefits, FAQs  
- WhatsApp integration for quick contact  

## 🇪🇸 Solución

### Flujo de reserva
- Proceso paso a paso  
- Indicación clara del progreso  
- Información transparente sobre precios y condiciones  
- Reducción de fricción en la toma de decisiones  

### Sistema UI
- Lenguaje visual consistente  
- Jerarquía clara para facilitar la lectura  
- Componentes reutilizables  

### Navegación y contenido
- Múltiples accesos al flujo de reserva  
- Secciones claras: tipos de vehículos, ofertas, beneficios, FAQs  
- Integración con WhatsApp para contacto rápido
